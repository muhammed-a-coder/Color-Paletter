# Color Paletter
A website where you can find cool and beautifull color palettes for your place.

✨ Technologies
---
- Figma

- Canva

- HTML

- CSS

🚀 Features
---
- Structured color system built with CSS variables for easy customization and scalability

- Each palette contains two primary colors and one supporting sub-color for practical design use

- Independent sections allow flexible updates without affecting other palettes

- Clean semantic HTML structure for maintainability and readability

- Responsive layout that adapts across different screen sizes

- Consistent typography and spacing to enhance visual balance

- Organized ID-based styling for precise control over each color block

- Clear hex code display for quick developer reference

📍 The Process
---

I built this color palette website to create something clean, structured, and actually useful rather than just a basic grid of colors. Each palette includes two main colors and a supporting sub-color, organized with semantic HTML and CSS variables for scalability and easy editing. The layout is responsive, and every section is independent, making the system simple, flexible, and practical for real design use.

## 🚦 Running the Project

1. Download the files and ensure they are in the same folder
2. Run index.html
